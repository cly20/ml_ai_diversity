# Supporting notes
This repository is in active development with the end goal of producing a 
resource for engaging a broad audience into data science, coding, and 
problem solving involving AI and ML. This codebase uses the streamlit, which 
facilitates the running and hosting of Web-based applications written in 
Python.

Streamlit supports embedding of markdown, images, text, multipage navigation, 
and hosting of external apps and webpages in embedded iframes.

# Proposed layout
While subject to change, Caley suggests the following rough layout:

1. Introduction/Welcome landing page
2. A context page with short summaries of what this tutorial covers
and in which sections
3. Intro to stats (hands on?)
4. Intro to data (big data, small data, data management and tidying)
5. Coding for AI and ML (hands on?)
6. Use cases of AI and ML (resource page)
7. Tutorials on AI and ML with clear outcomes
